## Description

Nest.js version of cadburythecat.com. Very simple GET routes showing a list of cat images and dynamic route parameters to show one image.

Hosted on Heroku

https://nest-cadbury.herokuapp.com/cats/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
